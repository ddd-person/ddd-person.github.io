Hermes Agent：一个会自我进化的 AI Agent，OpenClaw 的全面替代者

发表于2026-04-13分类于AI应用阅读次数

你有没有这种感觉：跟 AI 聊天，每次都像跟一个失忆的人对话？

上次你花了半小时跟它说清楚了你的项目背景、代码风格偏好、技术栈选择，结果关掉窗口再打开——它全忘了。你得从头再来一遍。

更烦的是，你在电脑上跟它聊到一半，想出门在手机上继续？不好意思，上下文不通，换个平台就是一个全新的对话。

今天介绍的 Hermes Agent，就是来终结这个痛苦的。它是目前我在 GitHub 上看到的最有野心的 AI Agent 项目——不是那种「你问我答」的聊天机器人，而是一个会自我进化、跨平台运行、越用越聪明的真正 Agent。

项目地址：https://github.com/NousResearch/hermes-agent
Stars：69.9k ⭐ | 开源协议：MIT | 作者：Nous Research

如果你之前用过 OpenClaw（圈子里叫"小龙虾"），那 Hermes Agent 可以看作它的全面升级版，而且官方直接内置了一键迁移功能。

一、Hermes Agent 是什么？

一句话概括：Hermes Agent 是一个会自我学习、自我进化的 AI 智能体。

它的 slogan 是 "The agent that grows with you"——和你一起成长的 Agent。

跟普通的 ChatGPT 或者 Claude 对话不同，Hermes Agent 是一个可以部署在你自己服务器上的独立 Agent。它有长期记忆、会自动学习新技能、可以跨平台（Telegram/Discord/微信等）跟你对话，而且不绑定任何一家 AI 模型——你想用 GPT 就用 GPT，想用 Claude 就切 Claude，一行命令的事。

来看看它跟普通 AI 聊天的区别：

维度普通 AI 聊天Hermes Agent
记忆关窗口就忘，每次从头来跨会话长期记忆，越用越懂你
平台只能在网页/APP 里用Telegram、Discord、Slack、命令行都行
学习能力不会变，用100次和用1次一样自动把经验沉淀为技能，越来越强
模型绑定绑死一家（GPT 或 Claude）随时切换，200+ 模型任选
部署位置厂商的服务器上你自己的机器上，数据全在你手里
自动化得你手动发消息才干活支持定时任务，自动在后台执行

二、核心能力——为什么说它会"自我进化"？

1. 闭环学习系统

这是 Hermes Agent 最核心的能力，也是它和其他 Agent 最大的区别。

当它完成一项复杂任务后，它会自动把这次的工作流总结成一个「技能文件」（Markdown 格式）。下次遇到类似任务，它不会从零开始摸索，而是直接调用之前沉淀的经验。

举个例子：

  • 第一次让它部署一个 Node.js 项目，它可能需要摸索一阵
  • 完成后，它自动把"部署 Node.js 项目"这个流程写成一个技能
  • 第二次让它部署，它直接调用这个技能，又快又稳

这就是所谓的"用得越多越聪明"——不是营销噱头，是真的有机制在背后跑。

2. 持久化记忆

Hermes Agent 用 SQLite 全文搜索 + LLM 总结机制来管理记忆。它会记住：

  • 你的技术栈偏好（"我用 React，不用 Vue"）
  • 你的沟通风格（"我喜欢简洁的回答，不要啰嗦"）
  • 之前对话的关键上下文

而且这些记忆是跨会话、跨平台的。你在命令行里跟它说了一遍你的偏好，切到 Telegram 上它依然记得。

3. 多平台通讯网关

这个功能简直太爽了。它内置统一网关，直接支持：

  • Telegram
  • Discord
  • Slack
  • WhatsApp
  • Signal
  • Email

也就是说，你可以在电脑上用命令行给它下任务，然后出门在 Telegram 上查看进度和结果。一个 Agent，所有渠道共享同一套技能和记忆。

4. 不锁定模型

一行命令切换 AI 模型：

hermes model

支持的模型源包括：

  • Nous Portal（Nous Research 自家的）
  • OpenRouter（200+ 模型随便选）
  • OpenAI（GPT 系列）
  • Anthropic（Claude 系列）
  • 以及你自己的本地模型端点

没有厂商锁定，没有代码改动，切模型就跟切频道一样简单。

5. 40+ 内置工具

Hermes Agent 开箱即用的工具集非常丰富：

  • 终端操作 & 文件管理
  • 网页搜索 & 浏览器自动化
  • 代码执行
  • 图像生成
  • 语音转文字
  • MCP 协议集成（连接各种外部 API）

6. 定时自动化（Cron）

支持自然语言设置定时任务。你可以跟它说：

"每天早上 9 点帮我整理一份 GitHub Trending 摘要发到 Telegram"

它就会在后台默默执行，到点给你推送结果。

三、跟 OpenClaw（小龙虾）比，强在哪？

很多人之前用的是 OpenClaw，那 Hermes Agent 为什么被称为它的"替代者"？

首先，Hermes Agent 官方直接提供了 OpenClaw 一键迁移功能：

# 交互式迁移
hermes claw migrate

# 预览会迁移什么
hermes claw migrate --dry-run

# 只迁移用户数据（不含密钥）
hermes claw migrate --preset user-data

会自动导入你在 OpenClaw 里的：

  • SOUL.md — 人格文件
  • 记忆 — MEMORY.md 和 USER.md
  • 技能 — 用户创建的技能文件
  • 命令白名单 — 审批规则
  • 消息配置 — 各平台的设置

设计上，Hermes Agent 相比 OpenClaw 的核心升级在于：闭环学习系统和Honcho 用户建模。前者让 Agent 能自动积累经验，后者让 Agent 在后台默默观察你的沟通风格和偏好，回复越来越"对味"。

四、怎么安装？一行命令搞定

快速安装

支持 Linux、macOS、WSL2：

curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

安装完之后：

# 重新加载 shell
source ~/.bashrc   # 如果用 zsh 则 source ~/.zshrc

# 启动 Hermes！
hermes

常用命令速查

命令用途
hermes启动交互式 CLI，开始对话
hermes model选择/切换 AI 模型
hermes tools配置启用哪些工具
hermes gateway启动消息网关（Telegram、Discord 等）
hermes setup运行完整的初始化向导
hermes update更新到最新版本
hermes doctor诊断问题

对话中的常用指令

指令用途
/new开始新对话
/model [provider:model]切换模型
/personality [name]切换人格
/skills查看已有技能
/retry重新生成回复
/compress压缩当前对话上下文
/usage查看 Token 用量
/insights [--days N]查看使用洞察

Android 用户

Hermes Agent 甚至支持在手机上通过 Termux 运行。具体安装方式参见官方 Termux 指南。

注意：Windows 原生不支持，需要先装 WSL2。

五、Hermes Agent 的部署灵活度

这一点也值得单独说。Hermes Agent 的运行环境极其灵活：

  • 最低配：一台 $5/月 的 VPS 就能跑
  • 最高配：GPU 集群、Serverless 基础设施（如 Modal、Daytona）
  • 本地：你的 Mac / Linux 笔记本
  • 手机：Android 通过 Termux

而且它不绑定你的电脑——你在服务器上跑着 Hermes Agent，人在外面通过 Telegram 给它发消息下任务，它在云端执行完把结果推给你。真正的「AI 管家」体验。

六、安全机制

让一个 AI Agent 在你的服务器上自主执行操作，安全自然是第一考虑。Hermes Agent 内置了几层防护：

  • 指令审批流：危险操作需要你手动确认
  • 危险模式拦截：自动识别和阻止高风险行为
  • 沙箱执行：支持 Docker、SSH、Singularity 等容器沙箱
  • 命令白名单：只有白名单内的操作才能自动执行

七、适合谁用？

  • 开发者/极客：想要一个常驻的 AI 助手，深度参与你的项目，而且不用每次重新交代背景
  • 自动化爱好者：需要把网页抓取、数据分析、定期报告等工作流自动化，而且希望这些流程越跑越成熟
  • 跨设备工作者：电脑上开发，手机上查看进度，一个 Agent 所有场景无缝衔接
  • OpenClaw 用户：如果你在用"小龙虾"，Hermes Agent 是它的官方精神续作，一键迁移无痛切换

九、总结

Hermes Agent 用一句话概括：

它是目前开源社区里最接近「个人 AI 管家」的项目——能记住你、能进化、能跨平台、不锁厂商、数据在你手里。

如果你之前用 OpenClaw，那现在可以无缝切换过来了。如果你还没用过任何 AI Agent，那 Hermes Agent 是一个非常好的起点——一行命令装好，开聊就行。
